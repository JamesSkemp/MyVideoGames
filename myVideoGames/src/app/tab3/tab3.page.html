Getting Started

In order to use MyVideoGames for your own collection you'll need to generate a valid XML file and make it available online. You can find instructions on how to do this on the project page at github.com/JamesSkemp/MyVideoGames.

Once the XML file has been created and is available online, you can update the Settings tab with the path to your file and then press Download File.

A sample file is included in the project, and all code is available on GitHub, at the link above.
